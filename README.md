📌 Attendance Management System Using QR Code and Camera
📖 Description

The Attendance Management System using QR Code and Camera is a digital solution designed to automate attendance tracking in educational institutions. It replaces traditional manual methods with a fast, secure, and efficient QR-based system.

Faculty generate a unique QR code for each lecture session, and students scan it using their device camera. Attendance is recorded instantly in a centralized database, ensuring accuracy and preventing proxy entries.

🚀 Features
📱 QR Code generation for each lecture
🔍 Camera-based QR code scanning
⏱️ Real-time attendance recording
🔐 Secure login (Admin / Faculty / Student)
❌ Prevention of duplicate & proxy attendance
🗄️ Centralized database management
📊 Automatic report generation
🛠️ Technologies Used
Frontend: Java (Swing / Android Studio)
Backend: MySQL / Firebase
Libraries: ZXing (QR Code generation & scanning)
Connectivity: JDBC
⚙️ How It Works
Faculty logs into the system
Generates a QR code for the lecture
Students scan the QR code using camera
System validates data
Attendance is stored in database instantly

🎯 Objective

To develop a secure and efficient attendance system that reduces manual work, improves accuracy, and prevents attendance fraud using QR technology.
